# WeatherVue3

Современное приложение прогноза погоды, созданное с использованием Vue 3 и Vite. Приложение предоставляет подробную информацию о погоде для городов по всему миру с использованием сервиса WeatherAPI.

## Особенности

- Прогноз погоды в реальном времени для выбранных городов
- Прогноз погоды на 3 дня
- Подробная статистика погоды (влажность, скорость ветра, вероятность дождя)
- Адаптивный пользовательский интерфейс с привлекательным дизайном
- Функция поиска городов
- Красивые иконки погоды в зависимости от условий
- Интерфейс на русском языке

## Используемые технологии

- [Vue 3](https://vuejs.org/) - Прогрессивный JavaScript фреймворк
- [Vite](https://vite.dev/) - Инструментарий для фронтенда нового поколения
- [WeatherAPI](https://www.weatherapi.com/) - Поставщик данных о погоде
- [CSS Variables](https://developer.mozilla.org/en-US/docs/Web/CSS/--*) - Для управления темой
- [Google Fonts](https://fonts.google.com/specimen/Montserrat) - Шрифт Montserrat для типографики

## Установка

1. Клонируйте репозиторий или перейдите в директорию проекта
2. Установите зависимости:

```bash
npm install
```

## Использование

Для запуска сервера разработки:

```bash
npm run dev
```

Для сборки в продакшен:

```bash
npm run build
```

Для предварительного просмотра продакшен-сборки:

```bash
npm run preview
```

## Конфигурация

Приложение использует WeatherAPI для получения данных о погоде. Ключ API жестко закодирован в файле `src/App.vue`. Чтобы использовать свой собственный ключ API:

1. Получите бесплатный ключ API на [WeatherAPI](https://www.weatherapi.com/)
2. Обновите параметр `key` в функции `getCity` в файле `src/App.vue`

## Обзор компонентов

- **PaneLeft**: Отображает основную информацию о погоде, включая температуру, состояние, дату и город
- **PaneRight**: Показывает статистику погоды и позволяет перемещаться между днями прогноза
- **DayCard**: Представляет отдельный день в прогнозе с температурой и иконкой погоды
- **Stat**: Отображает конкретные метрики погоды (влажность, ветер, вероятность дождя)
- **CitySelect**: Предоставляет интерфейс для изменения отображаемого города
- **Error**: Обрабатывает и отображает сообщения об ошибках

## Интеграция с API

Приложение интегрировано с WeatherAPI для получения:
- Текущих погодных условий
- Прогноза погоды на 3 дня
- Средних температур
- Уровня влажности
- Скорости ветра
- Вероятности дождя
- Условий и описаний погоды

## Дизайн-особенности

- Современный темный пользовательский интерфейс с градиентными акцентами
- Адаптивная верстка, оптимизированная для различных размеров экрана
- Пользовательские иконки погоды, которые меняются в зависимости от условий
- Плавные переходы между различными состояниями погоды
- Интуитивная навигация между днями прогноза

## Пользовательские директивы

Приложение включает пользовательскую директиву `v-focus`, которая автоматически фокусирует элементы ввода при их монтировании.
